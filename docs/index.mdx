# NEON CLI

Developed mit 💙 von NEON Software Solutions

---

Ein Command Line Interface für Dart, das nur ein Gas kennt.

## Installieren

🚨🚨🚨 Falls du noch die alte Version der CLI installiert hast (wenn du NEON im Terminal eintippst, passiert irgendwas), dann musst du sie erst deinstallieren! Das geht so: 🚨🚨🚨

```sh
onepub pub global deactivate neon_cli
```

Das Installieren geht mit meinem Lieblingsbefehl, dem Einzeiler:

```sh
onepub pub global activate neon_cli
```

### Was lass ich mich hier zwischen die Drehscheiben gravieren?

Einfach nur eine affengeile CLI 🐒

## Commands

### `NEON abfahrt`

Erstellt dir ein neues Flutter Projekt anhand der [NEON Core App][neon_core_app_link]. Dauert ungefähr 3.5 Tornados 🍺

### `NEON --help`

Für die Hilflosen unter uns.

### Eine kleine Übersicht

```sh
🥵 Ein absolutes High-Performer-Tool, vom wilden Typen für wilde Typen.

Usage: NEON <command> [arguments]

Global options:
-h, --help       Print this usage information.
    --version    Einmal die aktuelle Version to go bitte.

Available commands:
  abfahrt   NEON abfahrt <projektname>
            Erstellt ein neues Projekt mit dem angegebenen Namen nach NEON Maßstäben (to the 🌝) im aktuellen Verzeichnis.

Run "NEON help <command>" for more information about a command.
```

## Usage

```sh
# Erstelle eine neue App, die geile_app heißt:
NEON abfahrt geile_app
```

### Maintenance

Solltest du mal zu den glücklichen Seelen gehören, die ein Update für die CLI pushen, dann vergiss nicht, in der `pubspec.yaml` die Versionsnummer je nach Change (breaking, major, minor) zu ändern und anschließend `dart run build_runner build` laufen zu lassen, damit sich `lib/src/version.dart` automagisch aktualisiert! Dann pushst du das ganze auf deinen Feature-Branch, erstellst einen PR und stößst den Review-Prozess an!
